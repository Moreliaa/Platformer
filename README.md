A simple 2D platformer. WIP

# Controls
```
Up - Jump
Left, Right - Walk
(while airborne) A + directions - 8-directional boost
```
# Features
  * Updates at 60 fps
  * Gravity physics
  * Collision detection
  * Map Editor
  * Camera shake
  * Parallax Scrolling
# Libs
Uses LWJGL 3.1.6 ( https://github.com/LWJGL/lwjgl3/releases )
